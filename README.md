# AI-Powered Personal Finance Budget Chatbot

## Overview
A personal finance management system that helps users track expenses, compare spending with predefined budgets, and receive intelligent financial recommendations.

## Features
- Monthly income and expense tracking
- Category-wise spending analysis
- Budget vs Actual comparison
- Smart financial recommendations
- Finance chatbot
- Downloadable financial report

## Technologies Used
- Python
- Pandas
- Streamlit
- Matplotlib

## Dataset
- personal_transactions.csv
- Budget.csv

## How to Run

pip install -r requirements.txt

streamlit run app.py
