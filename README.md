# AI-Powered Personal Finance Budget Chatbot

## Project Overview

The AI-Powered Personal Finance Budget Chatbot is a Streamlit-based web application that helps users analyze their financial transactions, track expenses, compare spending against budgets, and receive smart financial recommendations.

The chatbot can answer finance-related questions based on transaction data and provide insights into income, expenses, savings, and spending patterns.

## Features

- Monthly Income, Expense, and Balance Summary
- Category-wise Spending Analysis
- Budget vs Actual Comparison
- Smart Financial Recommendations
- Interactive Finance Chatbot
- Data Visualization Dashboard
- Streamlit Cloud Deployment

## Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- Git & GitHub

## Dataset

The project uses a sample personal finance transaction dataset stored in CSV format.

Files:
